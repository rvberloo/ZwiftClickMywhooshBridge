# ZwiftClickMywhooshBridge
Dotnet Console to map zwift click buttons to keypresses recognised in mywhoosh

first version not fully stable, but does work .It maps Both K and I keys to clicks on the second Zwift Click (test with zwift click v2) controller as the first controller keeps losing the connection after approx a minute.

Plan to clean up the code and make the mapping more generic with a config file
